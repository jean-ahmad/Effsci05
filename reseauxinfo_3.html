Réseau informatique d'un hôpital

Le réseau informatique, dans l'image ci-contre, de l'hôpital est composé de 3 sous-réseaux, et il est connecté à internet. Les hôtes dans leurs sous-réseaux respectifs sont:

  • Sous-réseau 1: PC1, PC2, PC3, PC4, PC5.
  • Sous-réseau 2: PC6, PC7, PC8, PC9.
  • Sous-réseau 3: PC10, PC11, PC12, PC13.
  • Internet: serveur 1.

Le but de l'exercice est de compléter le réseau informatique de l'hôpital avec:

  • L'installation des adaptateurs réseau.
  • L'interconnexion des hôtes avec les appareils d'infrastructure.
  • La configuration des paramètres de réseau:

    • Adresse IPv4 et masque de sous-réseau,
    • SSID et Authentication pour les connexions sans-fil.

Les adresses des sous-réseaux en notation CIDR, le chiffre après l'adresse réseau représente le nombre de bits à 1 dans le masque de sous-réseau, sont:

  • Sous-réseau 1: 192.168.1.224/27
  • Sous-réseau 2: 192.168.2.64/26
  • Sous-réseau 3: 172.18.0.0/15
  • Internet: 8.1.84.0/22


Installation des documents de travail

Téléchargez le fichier hopital.zip, et sauvegardez-le sur votre clé USB ou dans votre espace de personnel.

Faites un clic droit sur le fichier téléchargé, et sélectionnez "extraire tout".

Faites un double clic gauche sur le dossier extrait, puis sur le fichier "hopital.pka" à l'intérieur du dossier.

Finalisation du réseau de l'hôpital

N'oubliez pas de sauvegarder régulièrement votre travail.

Transférez-le à la fin de chaque séance (lien vers le transfert du travail).



Installation des adaptateurs réseau et interconnexion:

Faites un clic gauche sur "Serveur 1", et constatez qu'il n'y a aucun module installé sur le serveur.



Faites un clic gauche sur "Cloud 1", et constatez qu'il y a deux modules, PT-CLOUD-NM-1CFE et PT-CLOUD-NM-1FGE, installés sur le cloud.

Le serveur étant à l'extérieur de l'hôpital, il est nécessairement connecté au cloud qui représente internet. Le réseau de l'hôpital étant, quant à lui, connecté au cloud par l'intermédiaire de la fibre optique, le serveur ne peut donc être connecté que sur le second module, liaison filaire Fast-Ethernet, du cloud.



Installez le module PT-HOST-NM-1CFE sur le serveur.

Connectez le serveur et le cloud avec un câble Ethernet droit.

Dans Packet Tracer, il y a généralement des voyants de liaison aux deux extrémités de la connexion qui relie deux appareils. Certaines connexions n'ont pas de voyants de liaison.

La couleur du voyant indique l'état de la liaison:

  • Vert clair: le média choisi est correct, et la liaison physique est établie.
  • Vert clignotant: des données sont échangées à travers le média.
  • Rouge: le média choisi est incorrect, et aucun signal ne peut être transmis.
  • Ambre: le port du commutateur est dans l'état bloqué à cause d'une boucle.

Les voyants de liaison sont les gros points en vert clair

Terminez l'installation des adaptateurs réseau et l'interconnexion en tenant compte des remarques suivantes:

  • Il est impossible de déplacer les hôtes ou les appareils d'infrastructure. Une boite de dialogue avec le message "Move Devices is locked" apparait si vous essayez de le faire.
  • Il est impossible de changer la configuration matérielle des appareils d'infrastructure, ainsi que la liaison par la fibre optique entre le routeur et le cloud. Une boite de dialogue avec le message "Add/Change Interfaces is locked" apparait si vous essayez de le faire.
  • Lors du raccordement d'un hôte au réseau de l'hôpital, il faut choisir l'appareil d'infrastructure le plus proche. Si le câble est trop long, aucun signal ne pourra être transmis, et les voyants de liaison apparaîtront en rouge.

Vérifiez votre score qui doit être, à la fin de cette étape, de 30/58.



Configuration des paramètres de réseau:

  1. Faites un clic gauche sur "PC1".
  2. Assurez-vous d'être, dans la fenêtre de configuration du PC qui s'ouvre, sur l'onglet "Desktop".
  3. Faites un clic gauche sur "IP Configuration".

  4. Faites un clic gauche, dans la zone "IP Configuration", sur le bouton "Static" permettant de configurer manuellement les paramètres de réseau, et modifiez les paramètres suivants:

    • L'adresse IPv4 dans la boite de dialogue en face du champ "IP Adress" (ex: 192.168.0.1).
    • Le masque de sous-réseau dans la boite de dialogue en face du champ "Subnet Mask" (ex: 255.255.255.0).


Terminez de configurer les paramètres de réseau, adresses IPv4 et masques de sous-réseau, pour tous les hôtes du réseau informatique de l'hôpital.

Vérifiez votre score qui doit être, à la fin de cette étape, de 58/58.



  1. Faites un clic gauche sur le "Point d'accès 1".
  2. Faites un clic gauche sur l'onglet "Config".
  3. Sélectionnez "Port 1" dans la zone de liste.


Les paramètres du point d'accès, pour connecter des hôtes sur le réseau sans fil, à relever sont:

  1. Le SSID (Service Set IDentifier) qui sert à identifier le réseau sans fil.
  2. Le mode de certification: WEP, WPA-PSK ou WPA2-PSK,
  3. La valeur de Key en mode WEP, ou celle de Pass Phrase en mode WPA-PSK ou WPA2-PSK,
  4. Le type de cryptage dans Encryption Type.


  1. Faites un clic gauche sur un PC à connecter au réseau sans fil.
  2. Faites un clic gauche sur l'onglet "Config".
  3. Sélectionnez "Wireless0" dans la zone de liste.


Reportez les paramètres du réseau sans fil, pour se connecter sur le "Point d'accès 1", sur le PC:

  1. Le SSID,
  2. Le mode de certification,
  3. La valeur de Key, ou celle de Pass Phrase,
  4. Le type de cryptage.


Quelques instants après avoir configurés les paramètres du réseau sans fil et s'ils sont corrects, une ligne hachurée, voir ci-contre, apparaît entre le point d'accès et le PC.

Terminez de configurer les paramètres du réseau sans fil sur les PCs qui le nécessitent.
